# 📊 Employee Attrition Predictor

An end-to-end Machine Learning application that predicts the likelihood of an employee leaving a company based on various HR metrics.

## ✨ Features
- **Machine Learning Pipeline**: Automated scaling and encoding using Scikit-learn.
- **Interactive UI**: Built with **Streamlit** for real-time risk analysis.
- **Data-Driven Insights**: Predicts attrition based on factors like Age, Department, and Salary.
- **Visualizations**: Displays top contributing factors for each prediction.

## 🛠️ Tech Stack
- **Language**: Python
- **Libraries**: Scikit-Learn, Pandas, Numpy, Streamlit
- **Model**: Random Forest Classifier
